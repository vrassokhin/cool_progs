Доказательство бинома Ньютона

Введение

Бином Ньютона - это основная формула в алгебре, которая выражает разложение степеней суммы двух слагаемых. Он назван в честь великого математика Исаака Ньютона.

Формула бинома Ньютона

Бином Ньютона выглядит следующим образом:

\((a + b)^n = \sum_{k=0}^{n} \binom{n}{k} a^{n-k} b^k\)

где \(\binom{n}{k} = \frac{n!}{k!(n-k)!}\)

Доказательство бинома Ньютона

Доказательство бинома Ньютона проводится методом математической индукции:

База индукции

Для \(n = 0\), \((a + b)^0 = 1\), что совпадает с правой частью формулы, так как \(\binom{n}{k} = 1\) только для \(k = 0\).

Индукционный шаг

\[(a + b)^n = \sum_{k=0}^{n} \binom{n}{k} a^{n-k} b^k\]

Предположим, что формула верна для \(n = m\):

\((a + b)^m = \sum_{k=0}^{m} \binom{m}{k} a^{m-k} b^k\)

Тогда для \(n = m + 1\) имеем:

\((a + b)^{m+1} = (a + b) (a + b)^m = a (a + b)^m + b (a + b)^m\)

Используя предположение индукции, получаем:

\(= a \sum_{k=0}^{m} \binom{m}{k} a^{m-k} b^k + b \sum_{k=0}^{m} \binom{m}{k} a^{m-k} b^k\)

Разделив сумму на два отдельных выражения, мы получаем:

\(= \sum_{k=0}^{m} \binom{m}{k} a^{m+1-k} b^k + \sum_{k=0}^{m} \binom{m}{k} a^{m-k} b^{k+1}\)

Преобразуя индексы в суммах и объединяя их, получаем:

\(= \sum_{k=0}^{m+1} \left( \binom{m}{k} + \binom{m}{k-1} \right) a^{m+1-k} b^k\)

Замечаем, что \(\binom{m}{k} + \binom{m}{k-1} = \binom{m+1}{k}\), что завершает доказательство.
