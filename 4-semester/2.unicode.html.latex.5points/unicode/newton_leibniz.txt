Формула Ньютона-Лейбница утверждает, что если f является непрерывной функцией на отрезке [a, b] и F является её первообразной, то

b
∫ f(x) dx = F(b) - F(a).
a

Доказательство:

Пусть f — непрерывная функция на [a, b], и пусть F — первообразная f, то есть F'(x) = f(x) для всех x ∈ [a, b].

По определению определённого интеграла:

b                n    
∫ f(x) dx = lim  ∑ f(x*_i)Δx_i ,
a           n→∞ i=1

где 
{[x_{i-1}, x_i]} — разбиение отрезка [a, b] на n равных частей, 
Δx_i = (b-a)/n, 
x*_i — произвольная точка в интервале [x_{i-1}, x_i].

Теперь рассмотрим разность F(x_i) - F(x_{i-1}). По теореме о среднем значении для интегралов существует такая точка c_i ∈ [x_{i-1}, x_i], что:

F(x_i) - F(x_{i-1}) = F'(c_i) (x_i - x_{i-1}).

Так как F'(x) = f(x), получаем:

F(x_i) - F(x_{i-1}) = f(c_i)Δx_i.

Суммируя по всем интервалам разбиения, получаем:

 n                       n
 ∑ F(x_i) - F(x_{i-1}) = ∑ f(c_i)Δx_i
i=1                      i=1

Левая часть этой суммы — это телескопическая сумма, которая сворачивается в:

F(b) - F(a).

Таким образом, имеем:
              n
F(b) - F(a) = ∑ f(c_i)Δx_i.
             i=1

Переходя к пределу при n→∞, мы видим, что правая часть выражения стремится к определённому интегралу:

b
∫ f(x) dx = F(b) - F(a).
a

Таким образом, формула Ньютона-Лейбница доказана:

b
∫ f(x) dx = F(b) - F(a).
a
